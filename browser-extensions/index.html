英文环境下常用的浏览器插件

2022-09-30

本文提到的所有插件均基于 chromium 内核在 Edge 上正常运行。持续多年的浏览器大战至今，Edge 依然在坚持与市占率第一的 Chrome 分羹，在经过多次迭代更新以后，用户体验不断增强，拓展商店资源也逐步丰富起来，达到了完全可用的程度。

我一直使用 Edge 的原因很简单，系统内置，界面简洁，功能丰富，UI 不会引起任何生理不适。也许大部分的人从来不会用到浏览器插件，这是一个听起来就让人头疼的东西，但目前对我来说，没有插件的浏览器，和刷牙没有牙刷是一个概念。

Workona Tab Manager

集成服务。整理，分区保存浏览器标签页，对于一次性打开几十个标签页且不想关闭的情况下非常有用。与 Edge 内置的 Add tad to group 和 鸡肋的 Collection 功能相比，简直是太好用了。

划词翻译

让查单词变得简单且轻松，包括 DeepL 多种翻译源。

Enhancer for YouTube

增强 Youtube 功能，我常用的有强制分辨率，广告去除。

Enjoy Learning Languages

提供双语字幕，字母大小位置可调节，准确率极高，与 Youtube 内置字幕识别不同的是，这款插件可以即时显示一句话的所有内容，从右侧字幕栏位置可以看到，其实所有字幕在打开视频的时候已全部生成完毕了。

Grammarly

英文语法纠正美化。Youtube 广告推荐，用户基数大，莫名的好用，写英文短篇小作文时帮了大忙。

系统工具类

GoFullPage

支持网页长截屏。

Vimium

从此上网只需键盘。

Wayback Machine

网页时光机。

IDM

download everythig.
